# **Краткий экскурс**
## 1. Файл mt6701_base.c содержит такие простые функции, как:
### a. настройка комм. интерфейса i2c;
### b. инициализация энкодера;
###   c. установка режима ABZ (если нужен инкрементный выход);
###   d. настройка 16 имп/оборот;
###   e. чтение угла от 0 до 360 градусов;
###   f. функцию для SPI (SSI) комм. интерфейса.
## 2. Файл mt6701_interface_template.c содержит такие функции, как:
###   a. запись байт в регистр энкодера;
###   b. использование HAL_I2C_Master_Transmit/Receive;
###   c. чтение байт из регистра энкодера.
## 3. Основной драйвер mt6701.c содержит такие функции, как:
###   a. настройка режимов, выхода, калбировки;
###   b. инициализация и чтение данных;
###   с. сохранение настроек в EEPROM и установка интерфейса I2C.

##   Работа:
###   1. mt6701_read_raw() читает 2 регистра:
####      a. 0x03 - старшие 8 бит угла;
####      b. 0x04 - младшие 6 бит угла.
###   2. Объединение в 14-битное значение;
###   3. Функция mt6701_read() конвертирует градусы.
